# TimesPro Brand Calendar

A customizable, moving-goalpost **Gantt-style brand calendar** for TimesPro brand activities
(Jun 2026 – Mar 2027). Built as a single, self-contained HTML file — no install, no build step.
Just open it in any browser.

## Open it

Double-click **`TimesPro_Brand_Calendar.html`** (or open it in any modern browser).
Both logos are embedded, and your changes auto-save to the browser.

## What it does

- **12 tentpole activities**, each a draggable block of stages. Drag a block to move the whole
  thing; every stage shifts with it ("moving goalpost").
- **Per-stage editing** — name, start date, duration (days), milestone/bar, tentative & weekly
  flags. Double-click any bar to edit it in place, or use the Customize panel.
- **Initiative owners**, exact dates on the chart, collapse/hide per track, weekend & today markers.
- **Versions** — save up to 2 named snapshots of the whole board and switch between them.
- **Export / Import JSON** and **Print → PDF**.
